WordPress Linked Articles Plugin Documentation

What did you get?

About

Thank you for purchasing the WordPress Linked Articles plugin.

This plugin allows you to easily link your articles/pages/posts and custom posts into unnamed groups.

You can manually control all connected posts for your WordPress blog, website or shop. You'll easily create sections of "Similar posts", "Post like this", "See also.." and others.

You do not need to create the hundreds unused tags no more, which would be used only twice or trice.

Search for article title, click on the found articles to group them. Then show linked articles automatically or manually anywhere in your theme.

Features

  • 13 layouts including sliders.
  • 8 appear animations.
  • 11 templates included.
  • Fully responsive.
  • Bootstrap compatible.
  • Highly customizable.
  • Compatible with all themes.
  • Works with WordPress Multisite.
  • Search by Custom Fields.
  • Optionally allows post editors to override output settings.
  • Easy to use shortcodes wizard.
  • Works with Custom Post Types.
  • Compatible with WooCommerce.
  • Sidebar Linked Articles Widget.
  • Fully translatable to any language using .po files.
  • Regular new features!

Installation

How to install?

Installation steps

Regular installation

  1. Get the package Ok, you got the plugin package :)
  2. Upload plugin package with WordPress http://yoursite.com/wp-admin/plugin-install.php?tab=upload or WordPress Multisite http://yoursite.com/wp-admin/network/plugin-install.php?tab=upload
  3. Activate WordPress Linked Articles Plugin on the WordPress Plugins page http://yoursite.com/wp-admin/plugins.php
  4. That's all! You can start from Quick start guide
  5. After plugin activation please go to plugin's settings on the page http://yoursite.com/wp-admin/options-general.php?page=wp-linked-articles to Set Up the plugin.

cPanel installation

  1. Upload plugin package through the cPanel File Manager to the /wp-content/plugins/
  2. Extract plugin package contents to the /wp-content/plugins/ folder
  3. Activate WordPress Linked Articles Plugin on the WordPress Plugins page http://yoursite.com/wp-admin/plugins.php
  4. That's all! You can start from Quick start guide
  5. After plugin activation please go to plugin's settings on the page http://yoursite.com/wp-admin/options-general.php?page=wp-linked-articles to Set Up the plugin.

FTP installation

  1. Unpack the plugin package with any archiving tool, which supports ZIP file format
  2. Upload files folder to the /wp-content/plugins/ to the server with any FTP client application, for example FileZilla
  3. Activate WordPress Linked Articles Plugin on the WordPress Plugins page http://yoursite.com/wp-admin/plugins.php
  4. That's all! You can start from Quick start guide
  5. After plugin activation please go to plugin's settings on the page http://yoursite.com/wp-admin/options-general.php?page=wp-linked-articles to Set Up the plugin.

Quick start guide

How to hit the ground running?

First of all you need to turn on Automatic Mode in the plugins settings.

You can find WordPress Linked Articles Plugin settings on the page http://yoursite.com/wp-admin/options-general.php?page=wp-linked-articles

Check the setting and click Save button. Let's use it now.


For Awesome Blog

Ok, you already have a lot of cool posts about fitness for example. It's time to show most relevant posts for them.

Open post editing page of one of the fitness posts, which you want to link among themselves. This is no matter which post of the whole set do you open.

On the post editing page find the Linked Articles adding widget


You don't have any linked articles yet. It's time to add them!

Type the part of the title of the post, which you want to add. Found posts were shown as a dropdown.

Navigate list by the keyboard or select by mouse. Selected article will be linked after post saving. Pick as more articles as you need. Remove articles, which are not needed anymore, by clicking X to the right of the titles.


For Great Shop

Ok, you have a WooCommerce shop and a lot of cool digital products for example. And you want to add Similar products section to the products page.

The process is similar to adding linked posts.

After adding linked products return to the Linked Articles adding widget on the post editing page.

Check the box Override settings for this product. The available settings will expand.

Change the label to the Similar products. Explanation of other settings you can find in the Advanced Usage section.


For Perfect Pages

Ok, you have the cool page about surfing for example. And you also have some posts about beaches and surfboards. let's assume, that you have already link some of them.

Open editing page of your Surfing page, navigate page contents and fine the best place, maybe the place where you mentioned beaches, where you surf.

Place cursor there and click the button of the Editor Linked Articles Shortcodes Wizard

Shortcode wizard popup will be opened. Search the post title of the linked posts in the Post field and then click Insert - linked articles shortcode will be placed.

You can place another shortcode for surfboards in different place of page content.

Save the page and preview your Perfect Page.

Set up

Where to start?

You can find WordPress Linked Articles Plugin settings on the page http://yoursite.com/wp-admin/options-general.php?page=wp-linked-articles

You can enable this plugin in two ways:

  1. Automatic
  2. Manual

Automatic

Automatic way is when you go to the settings page and turn on the automatic adding linked articles to the post content on a single post page.

For automatic mode you also can change ordering mode, posts limit to be shown on the single post page, layout and layout columns for linked articles output, appearing animation for static layouts and time_span options to limit posts by time span.

You can also disable automatic mode styles to replace them with your own and change the label.

Please see explanation of these settings in the "Are you developer?" section.
Manual

Or you could use it as shortcode [wp-linked-articles]. More information about shortcode below on Advanced Usage section and more information about WordPress Shortcode on WordPress.org.

Other settings

You can control post types, on which you want to show linked articles.

By default we search in articles titles only. If you want to search in article's content as well, please enable option below:

By default linked articles widget search is restricted to post type of the current post. Setting "Allow cross post type linking" will give you an ability to search and link articles with different post types.


You can control linked articles for any particular article/post/page at the post editing page in dashboard. This setting could be found at the post editor page.

Usage

How to use?

Adding linked articles

After you set up post types, for which plugin will be activated, you can go to the wordpress post editor. There you'll see widget of Linked Articles similar to this one:

If you'll need to customize output for this certain post, you can override plugin settings.

Start type the title (or the part of content, if you've turned on the setting "Search linked articles in content text") of the article which you want to link with the current one, then wait for search results dropdown.

You'll see the list of posts, matched your search criteria.

If the article, which you selected not grouped, it would be added to the list.

If the article, which you selected was already grouped with some other articles, then you'll see the wizard.

The wizard will allow you two options:

  • Merging groups of articles
    • if the current article have no group, then it will be added to the group of selected article.
    • if the current article have own group, then both groups will be merged into one group.
  • Moving article - in this case selected article will be moved from it's group, and will be added to the new group with the current article.
Changes to linked articles wouldn't be applied until you save your post. So feel free tp remove wrongly added linked articles and add another.

After articles were linked, you can unlink article by clikcing "Unlink this article" button on the post edit page

Mass Grouping linked articles
This operation cannot be undone, so please be careful.

You can group all or only not grouped articles from your categories. To group articles open the settings page, find "Categories Grouping".

Choose needed category and wait until wizard appears.

Wizard will allow you two options:

  • if there were some linked articles in category, then you could link all
  • or link only not linked articles
Linked articles shortcodes

You can add linked articles anywhere in your post. You can compose it manually or you can use a wizard for this.

Change options and click "Insert". You can select only one post for widget.

Linked articles sidebar widget

You can add linked articles to sidebar. Linked articles added to the sidebar wouldn't be actually linked. But will be shown all linked articles of all posts, which you'll select. Widget has the same options as shortcode.

Advanced Usage

Are you developer?

So you're developer. We take care about you, no matter, whether you get our plugin from the client or bought it for your purposes. You don't need to change anything inside the plugin (at least we hope so).

You will have full control on the plugin right from other plugin, or theme, or whatever you use in your wordpress.

Code control

You can override any of plugins classes: WPMemesEditor_Plugin, WPMemesEditor_Assets, WPMemesEditor_Frontend, somewhere before the after_setup_theme action.

Templates control

You can create automatic-archive-linked-articles.php, shortcode-archive-linked-articles.php, widget-linked-articles-view.php in your theme or copy from templates folder to your theme folder to override all linked articles template block under the post on the single page.

Fallback template for non-existent files is automatic-archive-linked-articles.php

You can also create templates for each article item of each layout and layout column pair for static and carousel layouts: content-linked-article-{layout name}-{layout-columns}.php; and templates for each other layout: content-linked-article-{layout name}.php

You can copy all plugin files from the templates plugin folder to your theme folder to override linked article template for each article.

Fallback template for non-existent files is content-linked-article.php

Plugin Output

Shortcode [wp-linked-articles] as well as linkedArticlesFrontController::disaply_linked_articles($atts = array()) has following parameters:

  • post_id (optional), if you wouldn't specify this parameter, the global post variable will be used.

  • ordering (optional), this option is shortcut for order and orderby options.

    Default value will be fetched from plugin's settings. You can override this parameter by custom order and orderby parameters.

    Allowed options are:

    • random - orderby=rand, order is empty
    • newest_first - orderby=post_date, order=desc
    • oldest_first - orderby=post_date, order=asc
  • order (optional), is the WP_Query order field. Default value will be fetched from plugin's settings.
  • orderby (optional), is the WP_Query orderby field. Default value will be fetched from plugin's settings.
  • posts_per_page (optional), is the WP_Query posts_per_page field. Default value will be fetched from plugin's settings. Use -1 or leave empty to get all.

  • layout (optional), layout for linked articles output.

    Default value will be fetched from plugin's settings. Leave empty to use static.

    Allowed options are:

    • static - simple responsive HTML view
    • slider - horizontal responsive slider with controls
    • vertical_slider - vertical responsive slider with controls
    • slideshow - horizontal responsive slider with controls and autoplay
    • slideshow - vertical responsive slider with controls and autoplay
    • carousel - responsive carousel with controls, number of items per slide is controled by posts_per_page option
    • ticker - responsive ticker

  • layout_columns (optional), layout columns for linked articles output (including Bootstrap columns or Pure.css columns).

    Default value will be fetched from plugin's settings.

    Allowed options are:

    • 1
    • 2
    • 3
    • 4
    • 6
  • label (optional). Default is "Linked Articles" for your localization.
  • appear_animation (optional).

    Default value will be fetched from plugin's settings. Leave empty to use without animation.

    Allowed options are:

    • bursting_bubbles
    • wind_from_west
    • wind_from_east
    • magnet
    • rain
    • melted_candy
    • signboard
    • barrier

  • time_span (optional), this option will limit linked articles by certain time span.

    Default value will be fetched from plugin's settings. Leave empty to use without time span.

    Allowed options are:

    • today
    • this_week
    • this_month
    • this_year
    • post_day
    • post_week
    • post_month
    • post_year
  • include_the_post (optional). If this option is specified and set to "true", then the article with post_id will be included to the linked articles.

You could call class method linkedArticlesFrontController::disaply_linked_articles($atts = array()) directly instead of using shortcode [wp-linked-articles].

You could manually get linked articles by using following code. 


$connected_post_ids = linkedArticlesModel::getConnectedPostsIds($post->ID);
if(!empty($connected_post_ids)) {
	
	// the query
	$query = new WP_Query(array(
	   'post_type' => get_post_types(array(
			'capability_type' => 'post',
			'public' => true
		)),
	   'post__in' =>  $connected_post_ids,
	   'orderby' => 'post__in'
	));
	
	// the loop
	if ( $query->have_posts() ) {
		foreach ($query->posts as $post) {
			$query->the_post();				
			echo $post->post_title;
		}
	} else {
		// no posts found
	}
	
	// restore original post data
	wp_reset_postdata();
}
